Getting Started with 11ty

I recently decided to start a personal blog and chose 11ty (Eleventy) as my static site generator. In this post, I'll walk through my experience setting it up and why I think it's a great choice for a personal blog.

Why 11ty?

11ty offers several advantages that made it appealing for my blog:

  • Simplicity: It's straightforward to set up and has minimal configuration
  • Flexibility: Works with multiple template languages
  • Speed: Generates static sites quickly and efficiently
  • No client-side JavaScript required: Results in fast, lightweight sites

The Setup Process

Getting started with 11ty was surprisingly easy. Here are the basic steps I followed:

  1. Installed 11ty via npm
  2. Created a basic configuration file
  3. Set up my folder structure for templates and content
  4. Added some basic styling

The most challenging part was deciding on the structure for my blog posts and how to organize the templates.

Using Markdown for Content

One of the things I love about 11ty is how easy it makes writing content in Markdown. I can focus on writing without worrying about HTML markup, while still having the flexibility to add custom HTML when needed.
